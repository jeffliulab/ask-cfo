### 决策

- **复制策略：rsync 选择性 + 个别 Edit**：避免直接整个 fin-pilot 拷过来引入大量 stock 特定代码。先 cp 通用文件 + mkdir 占位 dirs，再针对 stock-specific 的几个文件做内容重写
- **保留 fin-pilot 的 backend 三层 + 前端组件分层 100% 一致**：方便未来抽取 `agent-shell-ui` 共享包（v0.4+）
- **prompt 目录从 stock/ 改为 chat/（generic）**：CFO 暂时只有 chat 用途，后续模块如果有自己的 prompt 再各自建子目录（如 `bookkeeping/voucher_draft.j2`）
- **WorkspaceCard generic 而非 discriminated union**：fin-pilot 的 union 锁了 3 种 card_type；CFO 模块还在设计中，先用 generic dict payload，等模块上线再各自定义具体类型
- **LeftMenu 5 模块 2 active + 3 disabled**：让用户一眼看到完整愿景但能感知 v0.1 范围
- **legacy/wencfo/ 保留不删**：tax_service 的 Playwright 报税方案将来 v0.4 大概率要参考（界面老旧但流程对）；删了重写浪费
